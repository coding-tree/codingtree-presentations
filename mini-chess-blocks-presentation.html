Part 1: Draw the Board

⚑ when green flag clicked
set row to 0
repeat 5
set col to 0
repeat 5
if ((row + col) mod 2) = 0 then
set pen color to #EEEED2
else
set pen color to #769656
draw square
change col by 1
change row by 1

Part 2: Setup Each Piece

⚑ when green flag clicked
set isAlive to 1
set isSelected to 0
show
goToSquare 1 3

White King starts at row 1, column 3

♔ ♙ ♙ ♙ ♙
  • Each piece is a sprite
  • Track row & column
  • isAlive = in the game
  • isSelected = clicked

Part 2: Position Block

define goToSquare row col
set myRow to row
set myCol to col
go to x: -125 + ((col-1) * 50) + 25 y: 125 - ((row-1) * 50) - 25

This custom block converts board position to screen position!

Part 3: Click to Select

when this sprite clicked
if currentTurn = white then
if isAlive = 1 then
broadcast deselectWhite
set isSelected to 1
set selectedPiece to my name
switch costume to selected

Part 3: Deselect Others

when I receive deselectWhite
set isSelected to 0
switch costume to normal
  • All white pieces listen for this message
  • When received, they deselect themselves
  • Only one piece selected at a time!

Part 4: Detect Click Position

when stage clicked
set clickCol to round ((mouse x + 125) / 50) + 1
set clickRow to round ((125 - mouse y) / 50) + 1
if clickRow > 0 and clickRow < 6 then
if clickCol > 0 and clickCol < 6 then
broadcast squareClicked

Part 4: King Movement

when I receive squareClicked
if isSelected = 1 then
set rowDiff to abs(clickRow - myRow)
set colDiff to abs(clickCol - myCol)
if rowDiff < 2 and colDiff < 2 then
tryMove clickRow clickCol

King can move if distance is less than 2 squares!

Part 5: Capture Pieces

when I receive captureOnSquare
if isAlive = 1 then
if myRow = clickRow and myCol = clickCol then
set isAlive to 0
hide
if I am a King then
broadcast kingCaptured
♚ ❌

Capture the King to Win!

Part 5: Game Over

when I receive kingCaptured
set gameOver to 1
if currentTurn = white then
say White Wins! for 5 seconds
else
say Black Wins! for 5 seconds
broadcast gameEnd
