⚔️ Stwórz interfejs Collidable!
📌 1. Stwórz public interface Collidable
📌 2. Dodaj metody:
int getX();
int getY();
int getWidth();
int getHeight();
📌 3. Napisz metodę default:
default boolean collidesWith(Collidable other) {
  return getX() < other.getX() + other.getWidth()
    && getX() + getWidth() > other.getX()
    && getY() < other.getY() + other.getHeight()
    && getY() + getHeight() > other.getY();
}

🏆 Podsumowanie

  • ✅ Interfejsy — kontrakt, nie implementacja
  • ✅ implements — klasa spełnia obietnicę
  • ✅ Game Loop — input → update → render
  • ✅ deltaTime — płynny ruch niezależnie od FPS
  • ✅ GameWorld — kontener na encje (scene graph)
  • ✅ InputHandler — abstrakcja nad klawiaturą
  • ✅ Record — dane bez boilerplate
  • ✅ final — niemutowalność = bezpieczeństwo
