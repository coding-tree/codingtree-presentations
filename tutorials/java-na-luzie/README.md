# Java na Luzie

**Od Pythona, przez Jave, az po LWJGL i wlasnego Minecrafta!**

Kurs sklada sie z 4 sesji, ktore przeprowadza Cie przez najwazniejsze elementy Javy.
Kazda sesja to praktyczny projekt, ktory przyblizy Cie do budowania wlasnej gry 3D
z uzyciem LWJGL (Lightweight Java Game Library).

## Sesje

### Sesja 1: Stworz Swoj Swiat
Podstawy programowania obiektowego w Javie. Tworzymy klasy Entity, Player, Monster
i Dragon, a nastepnie budujemy Arene, na ktorej walcza nasi bohaterowie.

### Sesja 2: Ekwipunek Bohatera
Generyki, kompozycja i enumy. Projektujemy system przedmiotow (Items), ekwipunek
(Inventory) i mechanike craftingu.

### Sesja 3: Silnik Gry
Interfejsy, petla gry i architektura silnika. Implementujemy GameWorld, game loop
i system zarzadzania stanami gry.

### Sesja 4: Supermoc Javy
Nowoczesna Java w praktyce: logowanie z SLF4J, wyrazenia lambda, Stream API
i Optional. Refaktoryzujemy nasz projekt uzywajac najlepszych praktyk.

## Wymagania

- Znajomosc Pythona (lub innego jezyka programowania)
- IntelliJ IDEA (Community Edition wystarczy)
- Maven (zintegrowany z IntelliJ)
- Java 17+

## Uwagi

- **Jezyk kursu:** polski
- **Jezyk kodu:** angielski (nazwy klas, metod, zmiennych)
- **Cel koncowy:** po ukonczeniu kursu bedziesz gotowy, zeby zaczac przygode
  z LWJGL i budowaniem wlasnego klona Minecrafta!
