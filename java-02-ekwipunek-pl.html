🎒 Inventory<T extends Item>

public class Inventory<T extends Item> { private List<T> items = new ArrayList<>(); public void add(T item) { items.add(item); } public void remove(T item) { items.remove(item); } public List<T> getItems() { return Collections.unmodifiableList(items); } }
🎮 Ekwipunek
T = dowolny typ dziedziczący po Item. Kompilator pilnuje!
