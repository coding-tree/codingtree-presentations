Liczenie binarne

Każda pozycja ma swoją wartość - jak w naszych liczbach!

Każdy bit (cyfra binarna) ma wartość:

8
4
2
1

Jeśli bit jest włączony (1) - dodajemy jego wartość
Jeśli jest wyłączony (0) - nie dodajemy


Na przykład: 0101 = 0 + 4 + 0 + 1 = 5

Kliknij bity, żeby tworzyć liczby!
